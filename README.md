Logical Gate SimulatorUn simulateur 

interactif de portes logiques développé en HTML/CSS/JavaScript.
L’objectif du projet est d’apprendre à utiliser des script en js


Présentation du projet

Ce projet permet à l’utilisateur de :
- sélectionner une porte logique
- afficher automatiquement la table de vérité correspondante
- modifier les entrées à l'aide de checkbox
- visualiser la sortie de la porte logique
- afficher l’image de la porte sélectionnée
  
.La zone de simulation est générée dynamiquement via JavaScript pour s’adapter à la porte choisie.
